# 5G MU-MIMO System Simulation using GME and Neural Network

## 📌 Project Description
This project focuses on the simulation of a 5G MU-MIMO (Multi-User Multiple Input Multiple Output) system using advanced techniques such as GME-based power allocation and Neural Network prediction. The aim is to improve system performance, efficiency, and signal quality.

## 🎯 Objectives
- To simulate a 5G MU-MIMO communication system  
- To implement GME-based power allocation  
- To apply Neural Network for prediction  
- To analyze system performance using different metrics  

## 🚀 Features
- Efficient power allocation using GME  
- Intelligent prediction using Neural Network  
- MRT (Maximum Ratio Transmission) beamforming  
- Performance comparison using graphs  

## 🛠️ Technologies Used
- MATLAB / Python  
- Machine Learning  
- Wireless Communication Concepts  

## 📊 Output / Results
- Graphs showing system performance  
- Improved spectral efficiency  
- Better signal quality and reduced interference  

## 📁 Files Included
- Source code files (.m / .py)  
- Simulation output graphs  
- Project report (PDF/Word)
